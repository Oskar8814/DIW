Estilos CSS: los ID

Dentro de la hoja se pueden aplicar estilos a etiquetas muy concretas a través del atributo ID aunque lo habitual es usar clases.

Etiquetas <p> con distinto ID de estilo

En el fichero CSS un estilo para los ID se define con un nombre precedido por una almohadila como por ejemplo: #definicion { propiedad1: valor1; propiedad2: valor2; }

Este es el párrafo con el estilo #parrafo_especial especificado en la hoja de estilos. Se aplica con el atributo id.
